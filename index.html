Supabase 연동

데이터를 Supabase (PostgreSQL)에 실시간 저장합니다

push_pin연동 방법

1. supabase.com에서 프로젝트 생성 → URL과 anon key 입력
2. SQL Editor에서 스키마 및 RLS 정책 실행
3. Authentication → Providers에서 Google OAuth 활성화
4. Google Cloud Console에서 OAuth 클라이언트 ID 발급 → 아래 입력
5. 저장 후 재접속하면 Google 로그인 화면으로 전환됩니다

mail이메일 발송 관리 (Resend)

Resend API를 통해 매칭/세션 알림 이메일을 자동 발송합니다.

push_pinResend 설정 방법

1. resend.com 회원가입 (무료 플랜: 월 3,000건)
2. API Keys에서 키 발급
3. Supabase Dashboard → Edge Functions → send-email → Secrets에 추가:
RESEND_API_KEY = re_xxxxxxxxx
4. 커스텀 도메인 연결 시 FROM_EMAIL도 설정:
FROM_EMAIL = 멘토링 허브 <noreply@yourdomain.com>
