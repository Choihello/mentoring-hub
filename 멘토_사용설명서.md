# 멘토링허브 멘토 사용설명서

---

## 1. 로그인

1. 멘토링허브 접속
2. **Google로 로그인** 버튼 클릭
3. 멘토로 등록된 Google 계정 선택
4. 자동으로 멘토 화면 진입

---

## 2. 사이드바 메뉴 구성

| 순서 | 메뉴 | 설명 |
|------|------|------|
| 1 | 매칭 알림 | 새로운 매칭 제안 확인 및 수락/거절 |
| 2 | 가용 일정 등록 | 멘토링 가능한 시간 등록 |
| 3 | 예약 현황 | 멘티 신청 확인, 세션 관리 |
| 4 | 멘토링 일지 | 일지 작성 및 조회 |
| 5 | 공지사항 | 관리자 공지 확인 |
| 6 | 요청하기 | 관리자에게 문의/요청 |

> 매칭 알림, 공지사항, 요청하기 메뉴에는 미확인 항목이 있을 때 숫자 배지가 표시됩니다.

---

## 3. 매칭 알림

관리자가 멘티와의 매칭을 제안하면 이 화면에서 확인합니다.

### 3-1. 새로운 매칭 제안

매칭 제안이 도착하면 다음 정보가 표시됩니다:
- 멘티 이름
- 희망 분야, 희망 일정
- 요청사항 (멘티가 작성한 내용)
- 담당자 메모 (관리자가 작성한 참고사항)

**수락하기:**
1. **수락** 버튼 클릭
2. 매칭 확정 안내 팝업 표시
3. **가용 일정 등록** 페이지로 이동 안내

> 수락하면 멘토와 멘티 모두에게 확정 이메일이 발송됩니다.

**거절하기:**
1. **거절** 버튼 클릭
2. 확인 후 거절 처리
3. 관리자에게 거절 알림 이메일 발송 (재매칭 진행)

### 3-2. 매칭 이력

과거 매칭 제안의 처리 이력을 확인합니다.
- 상태: 확정 / 거절 / 해제
- 확정 후 가용 일정이 없으면 등록 안내 메시지 표시

---

## 4. 가용 일정 등록

멘티가 예약할 수 있는 시간을 등록하는 화면입니다.

### 4-1. 안내 배너

매칭된 멘티가 있으나 등록된 가용 일정이 없으면 상단에 안내 배너가 표시됩니다.
- 대기 중인 멘티 이름 표시
- **+ 시간 추가** 버튼으로 바로 등록 가능

### 4-2. 달력

- 월별 달력에서 날짜를 클릭하여 일정을 등록합니다
- 날짜 색상 의미:
  - **연한 색** : 가용 슬롯이 등록된 날
  - **진한 색** : 예약이 확정된 슬롯이 있는 날
  - **회색** : 지난 날짜 (등록 불가)
- 좌우 화살표로 월 이동

### 4-3. 일정 등록 방법

1. 달력에서 **원하는 날짜 클릭** (또는 **+ 시간 추가** 버튼)
2. 등록 팝업에서:
   - **날짜** 확인 (선택한 날짜가 자동 입력)
   - **장소** 입력 (예: "공학관 301호", "Zoom", "카페")
   - **시간 선택** : 09:00~22:00 중 원하는 시간 버튼 클릭 (복수 선택 가능)
3. **일정 추가** 클릭

> 하나의 날짜에 여러 시간을 동시에 등록할 수 있습니다.
> 당일인 경우 이미 지난 시간은 선택할 수 없습니다.

### 4-4. 등록된 일정 관리

달력 아래에 등록된 일정 목록이 날짜별로 표시됩니다.

- **가능** : 멘티가 예약할 수 있는 상태
- **예약됨** : 멘티가 이미 예약한 상태

각 슬롯에서 가능한 작업:
- **수정** : 시간/장소 변경
- **삭제** : 슬롯 제거 (예약되지 않은 경우만)

## 5. 예약 현황

멘티의 세션 신청을 확인하고 관리하는 화면입니다.

### 5-1. 일정 등록 리마인더

매칭 확정 후 24시간이 지났는데 가용 일정이 없으면 상단에 알림이 표시됩니다.
**등록하러 가기** 버튼으로 일정 등록 페이지로 이동합니다.

### 5-2. 신청 대기 (왼쪽)

멘티가 새로 신청한 세션 목록입니다.

표시 정보:
- 멘티 이름 (팀)
- 날짜, 시간, 장소
- 멘토링 주제 (멘티가 입력한 요청사항)

**수락:**
- **수락** 클릭 → 세션 확정 → 멘티에게 확정 이메일 발송
- 상태가 '확정'으로 변경

**거절:**
- **거절** 클릭 → 세션 취소 → 슬롯이 다시 '가능' 상태로 복원
- 멘티에게 취소 이메일 발송

### 5-3. 확정 세션 (왼쪽)

수락하여 확정된 예정 세션 목록입니다.

- **완료 처리** : 멘토링을 실시한 후 클릭 → 상태가 '완료'로 변경
- **취소** : 불가피한 사유로 취소 → 슬롯 복원, 멘티에게 취소 이메일 발송

### 5-4. 완료 세션 (오른쪽)

완료된 세션 목록입니다.

- 일지 미작성 시: **일지 작성** 버튼 표시
- 일지 작성 완료 시: **일지 보기** 버튼 표시
- 멘티 피드백이 있으면 평가 점수 표시

---

## 6. 멘토링 일지

### 6-1. 작성 대기 (왼쪽)

완료되었으나 일지를 아직 제출하지 않은 세션이 표시됩니다.
**일지 작성** 클릭으로 작성 팝업을 엽니다.

### 6-2. 일지 작성 방법

1. **일지 작성** 버튼 클릭
2. 작성 팝업에서 다음 항목 입력:
   - **멘토링 유형** : 드롭다운에서 선택
   - **소요 시간** : 실제 멘토링 시간 입력
   - **주요 상담 내용** : 멘토링에서 다룬 핵심 내용
   - **멘티 현황 및 문제점** : 멘티의 현재 상태, 어려움
   - **다음 멘토링 계획** : 다음에 진행할 내용
   - **종합 평가** : 전반적인 평가
   - **사진 첨부** : 필요 시 첨부 (선택)
3. **일지 저장** 클릭

### 6-3. 제출 완료 (오른쪽)

이미 제출한 일지 목록입니다.
- **상세 보기** : 일지 전체 내용 확인
- **수정** : 일지 내용 수정

> 일지는 관리자가 열람할 수 있으며, 미제출 시 관리자에게 미제출 목록으로 표시됩니다.

---

## 7. 공지사항

관리자가 등록한 공지사항을 확인합니다.

- **전체** 대상 공지와 **멘토만** 대상 공지가 표시됩니다
- 중요 공지는 상단에 고정되며 느낌표 아이콘이 표시됩니다
- 클릭하면 읽음 처리됩니다

---

## 8. 요청하기

관리자에게 문의사항이나 요청을 보낼 수 있습니다.

### 8-1. 요청 작성

1. **요청 유형** 선택 : 일정변경 / 멘토변경 / 기타문의
2. **제목** 입력
3. **내용** 입력
4. **제출하기** 클릭

### 8-2. 요청 현황

제출한 요청의 처리 상태를 확인합니다.

- **대기중** : 관리자 검토 전
- **처리중** : 관리자가 회신함
- **처리완료** : 요청 처리 완료

관리자 회신이 있으면 요청 하단에 회신 내용이 표시됩니다.

---

## 9. 전체 워크플로우 요약

```
[멘토 활동 흐름]

1. 매칭 알림 확인
   └─ 새 매칭 제안 → 수락 또는 거절

2. 가용 일정 등록 (수락 후)
   └─ 달력에서 날짜 선택 → 시간/장소 등록

3. 예약 현황에서 멘티 신청 확인
   └─ 수락 → 세션 확정
   └─ 거절 → 슬롯 복원

4. 멘토링 실시 후 완료 처리
   └─ 확정 세션에서 "완료 처리" 클릭

5. 멘토링 일지 작성
   └─ 완료된 세션에 대해 일지 제출

6. 공지사항 확인 / 요청사항 제출
```

### 이메일 알림을 받는 경우

| 시점 | 내용 |
|------|------|
| 매칭 제안 시 | 새 매칭 제안 안내 |
| 매칭 확정 시 | 매칭 확정 안내 |
| 멘티가 세션 신청 시 | 새 세션 신청 알림 |
| 멘티가 세션 취소 시 | 세션 취소 안내 |
| 관리자가 세션 취소 시 | 세션 취소 안내 |

> 알림 이메일은 관리자가 등록한 알림 이메일 주소로 발송됩니다.
> 알림 이메일이 별도 등록되지 않은 경우 로그인 이메일로 발송됩니다.
