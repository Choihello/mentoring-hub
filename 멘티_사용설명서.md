# 멘토링허브 멘티 사용설명서

---

## 1. 로그인

1. 멘토링허브 접속
2. **Google로 로그인** 버튼 클릭
3. 멘티로 등록된 Google 계정 선택
4. 자동으로 멘티 화면 진입

---

## 2. 사이드바 메뉴 구성

| 순서 | 메뉴 | 설명 |
|------|------|------|
| 1 | 매칭 신청 | 멘토 매칭 신청 및 현황 확인 |
| 2 | 멘토 예약하기 | 매칭된 멘토의 가용 일정에서 세션 예약 |
| 3 | 내 멘토링 내역 | 신청/확정/완료 세션 확인 및 피드백 작성 |
| 4 | 공지사항 | 관리자 공지 확인 |
| 5 | 요청하기 | 관리자에게 문의/요청 |

> 매칭 신청, 공지사항, 요청하기 메뉴에는 미확인 항목이 있을 때 숫자 배지가 표시됩니다.

---

## 3. 매칭 신청

멘토링을 받기 위해 멘토 매칭을 신청하는 화면입니다.

### 3-1. 확정된 매칭

이미 매칭이 확정된 멘토가 있으면 화면 상단에 표시됩니다.
- 멘토 이름, 전문 분야
- 확정일
- 담당자 메모 (있는 경우)

### 3-2. 매칭 신청 방법

1. **희망 분야** 선택 (복수 선택 가능)
   - 마케팅 / 투자유치 / 기술개발 / 해외진출 / 재무회계 / 법률 / 기타
2. **희망 일정** 선택 (복수 선택 가능)
   - 평일 오전 / 평일 오후 / 주말
3. **요청사항** 입력 (멘토링에 대한 구체적인 희망사항)
4. **매칭 신청하기** 클릭

> 매칭 신청은 기존 매칭 유무와 관계없이 언제든 추가 신청할 수 있습니다.

### 3-3. 신청 현황

제출한 매칭 신청의 처리 상태를 확인합니다.

| 상태 | 의미 |
|------|------|
| 대기중 | 관리자 검토 전 |
| 제안됨 | 관리자가 멘토에게 제안함, 멘토 응답 대기 중 |
| 확정 | 멘토가 수락하여 매칭 완료 |
| 종료 | 관리자에 의해 매칭이 해제/종료됨 |

---

## 4. 멘토 예약하기

매칭된 멘토의 가용 일정에서 멘토링 세션을 예약합니다.

### 4-1. 멘토 선택 (왼쪽)

매칭이 확정된 멘토 중 가용 일정이 있는 멘토가 목록에 표시됩니다.
- 멘토 이름, 전문 분야, 소속
- 예약 가능한 슬롯 수 (녹색 배지)
- 클릭하여 선택

> 매칭되지 않은 멘토는 목록에 표시되지 않습니다.
> 매칭은 되었으나 멘토가 아직 일정을 등록하지 않은 경우 "현재 예약 가능한 멘토가 없습니다"로 표시됩니다.

### 4-2. 시간 선택 (오른쪽)

멘토를 선택하면 오른쪽에 가용 시간이 날짜별로 표시됩니다.

각 시간 버튼에 표시되는 정보:
- **시간** (예: 14:00)
- **장소** (예: 공학관 301호)
- **예약 가능** 표시

### 4-3. 예약 방법

1. 왼쪽에서 **멘토 선택**
2. 오른쪽에서 **원하는 시간 클릭**
3. 예약 팝업에서:
   - 선택한 멘토, 날짜, 시간, 장소 확인
   - **요청 사항** 입력 (10자 이상, 멘토링 받고 싶은 주제나 내용)
4. **예약 신청하기** 클릭
5. 신청 완료 → 자동으로 **내 멘토링 내역** 페이지로 이동

> 신청 즉시 멘토에게 이메일 알림이 발송됩니다.
> 멘토가 수락해야 세션이 확정됩니다.

---

## 5. 내 멘토링 내역

신청한 세션의 진행 상태를 확인하고 관리하는 화면입니다.

### 5-1. 진행 중 (왼쪽)

현재 진행 중인 세션이 표시됩니다.

**대기 상태 (멘토 확인 대기 중):**
- 멘토 이름, 분야
- 날짜, 시간, 장소
- 멘토링 주제
- "멘토 확인 대기 중입니다" 안내 문구
- **신청 취소** 버튼 (취소 시 슬롯 복원)

**확정 상태 (멘토가 수락함):**
- "멘토링이 확정되었습니다!" 안내 문구
- 장소/방법 안내
- **신청 취소** 버튼 (불가피한 경우)

> 일정 변경이 필요하면 신청을 취소한 후 다시 예약하세요.
> 세션 취소 시 멘토에게 이메일 알림이 발송됩니다.

### 5-2. 완료 (오른쪽)

멘토링이 완료된 세션 목록입니다.

- 피드백 미작성 시: **피드백 작성** 버튼 표시
- 피드백 작성 완료 시: "피드백 완료" 표시

---

## 6. 피드백 작성

멘토링 완료 후 멘토에 대한 피드백을 작성합니다.

### 6-1. 작성 방법

1. **내 멘토링 내역** → 완료 세션에서 **피드백 작성** 클릭
2. 피드백 팝업에서:
   - **멘토링의 도움이 된 점** 입력
   - **멘토의 개선 바라는 점** 입력 (선택)
   - **멘토 평가** 선택
3. **피드백 제출** 클릭

> 피드백은 관리자가 열람할 수 있으며, 멘토의 세션 완료 화면에도 평가 점수가 표시됩니다.

---

## 7. 공지사항

관리자가 등록한 공지사항을 확인합니다.

- **전체** 대상 공지와 **멘티만** 대상 공지가 표시됩니다
- 중요 공지는 상단에 고정되며 느낌표 아이콘이 표시됩니다
- 클릭하면 읽음 처리됩니다

---

## 8. 요청하기

관리자에게 문의사항이나 요청을 보낼 수 있습니다.

### 8-1. 요청 작성

1. **요청 유형** 선택 : 일정변경 / 멘토변경 / 기타문의
2. **제목** 입력
3. **내용** 입력
4. **제출하기** 클릭

### 8-2. 요청 현황

제출한 요청의 처리 상태를 확인합니다.

| 상태 | 의미 |
|------|------|
| 대기중 | 관리자 검토 전 |
| 처리중 | 관리자가 회신함 |
| 처리완료 | 요청 처리 완료 |

관리자 회신이 있으면 요청 하단에 회신 내용이 표시됩니다.

---

## 9. 전체 워크플로우 요약

```
[멘티 활동 흐름]

1. 매칭 신청
   └─ 희망 분야 + 일정 선택 → 신청 제출
   └─ 관리자가 멘토 제안 → 멘토 수락 시 매칭 확정

2. 멘토 예약하기 (매칭 확정 후)
   └─ 매칭된 멘토 선택 → 가용 시간에서 예약
   └─ 요청사항 입력 → 신청 완료

3. 멘토링 진행
   └─ 멘토가 수락하면 세션 확정
   └─ 확정된 일시/장소에서 멘토링 진행

4. 피드백 작성 (멘토링 완료 후)
   └─ 내 멘토링 내역 → 완료 세션 → 피드백 작성

5. 추가 멘토링 필요 시
   └─ 같은 멘토: 멘토 예약하기에서 추가 예약
   └─ 다른 멘토: 매칭 신청에서 추가 신청
```

### 이메일 알림을 받는 경우

| 시점 | 내용 |
|------|------|
| 매칭 확정 시 | 멘토와의 매칭 확정 안내 |
| 멘토가 세션 수락 시 | 세션 확정 안내 (일시, 장소) |
| 멘토가 세션 취소 시 | 세션 취소 안내 |
| 관리자가 세션 취소 시 | 세션 취소 안내 |

## 10. 자주 묻는 질문

**Q. 매칭 신청 후 얼마나 기다려야 하나요?**
관리자가 검토하여 적합한 멘토에게 제안합니다. 멘토가 수락하면 매칭이 확정됩니다. 구체적인 소요 시간은 관리자에게 문의하세요.

**Q. 매칭된 멘토가 마음에 들지 않으면 어떻게 하나요?**
요청하기 메뉴에서 '멘토변경' 유형으로 요청을 제출하세요.

**Q. 예약 가능한 시간이 없으면 어떻게 하나요?**
멘토가 아직 가용 일정을 등록하지 않은 상태입니다. 잠시 후 다시 확인하거나, 요청하기 메뉴에서 관리자에게 문의하세요.

**Q. 예약한 세션의 일정을 변경하고 싶어요.**
현재 세션을 취소한 후 다른 시간으로 다시 예약하세요. 또는 요청하기에서 '일정변경' 요청을 제출하세요.

**Q. 멘토링을 추가로 더 받을 수 있나요?**
네. 같은 멘토에게 추가 예약하거나, 매칭 신청에서 다른 분야의 멘토를 추가로 신청할 수 있습니다. 횟수 제한 없이 자유롭게 신청 가능합니다.
